# 🌱 Grow & Thrive

A responsive web application designed to help unemployed and low-income individuals grow their own food and potentially start small businesses selling or cooking vegetables.

## 📱 Features

### 🔐 Authentication System
- **Sign Up**: Create an account with first name, last name, age, email, and password
- **Login**: Secure login with email and password validation
- **Password Requirements**: Must be 8+ characters with uppercase letter and special character
- **Welcome Screen**: Personalized greeting after successful login

### 🏠 Home Page
- **Vegetable Gallery**: Browse all available vegetables with images and descriptions
- **Search Functionality**: Filter vegetables by name
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop

### 🌿 Vegetable Detail Pages
Each vegetable includes:

#### 🌱 Soil Suitability Check
- Interactive questionnaire about soil type, rainfall, and fertilizer access
- Real-time suitability assessment
- Personalized recommendations for soil improvement

#### 🛠️ Resources & Tools
- Complete list of required tools (garden trowel, stakes, etc.)
- Seed requirements
- Compost and fertilizer needs

#### 📋 Planting Instructions
- Step-by-step planting guide
- Detailed descriptions for each step
- Video tutorial placeholder (YouTube integration ready)

#### 📅 Growth Tracker
- Interactive calendar showing growth timeline
- Daily task checklist
- Progress tracking from planting to harvest
- Start tracking functionality to monitor your plants

#### 🍽️ Post-Harvest Options
- **Marketplace**: List your vegetables for sale with pricing
- **Recipes**: Find delicious recipes using your harvested vegetables

### 🛒 Marketplace
- List vegetables for sale with quantity, price, and contact info
- Browse available vegetables from other users
- Simple and intuitive selling process

### 👨‍🍳 Recipe Section
- Collection of recipes for each vegetable
- Difficulty levels and cooking times
- Complete ingredient lists and instructions

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software installation required

### Installation
1. Download all files to a folder on your device
2. Open `index.html` in your web browser
3. The application will load immediately

### First Time Use
1. **Welcome Screen**: Read about the app's purpose and benefits
2. **Sign Up**: Create your account with your details
3. **Browse Vegetables**: Explore the vegetable gallery
4. **Choose a Vegetable**: Click on any vegetable to learn more
5. **Start Growing**: Follow the step-by-step guides
